深入开展师德建设主题教育活动，争做"四有"好老师

活动概述

为深入学习贯彻习近平新时代中国特色社会主义思想，落实立德树人根本任务，大数据学院于2024年3月隆重启动"师德建设主题教育月"活动。本次活动以"立德树人，培根铸魂"为主题，旨在进一步提升教师队伍师德师风建设水平，引导广大教师自觉践行教书育人使命。

在启动仪式上，学院领导深入阐述了新时代教师职业道德规范的重要内涵，强调教师要以德立身、以德立学、以德施教。与会教师围绕如何加强师德修养、提升育人能力展开深入研讨，现场气氛热烈。通过交流分享，教师们纷纷表示要将师德建设内化于心、外化于行，争做"四有"好老师。

活动期间，学院将通过专题讲座、案例研讨、师德承诺等多种形式，引导教师深入领会习近平总书记关于教育的重要论述，牢记为党育人、为国育才使命，坚持教书与育人相统一、言传与身教相统一、潜心问道与关注社会相统一、学术自由与学术规范相统一。

本次主题教育月活动的开展，标志着我院师德师风建设进入新阶段。学院将以此为契机，持续深化师德建设长效机制，推动形成优良教风学风，为培养德智体美劳全面发展的社会主义建设者和接班人作出新的更大贡献。

大数据学院作为我校新兴的重点院系，始终将师德师风建设作为教师队伍建设的首要任务。近年来，学院教师在教学、科研等方面取得了显著成绩，多人获得省级优秀教师、教学能手等荣誉称号。本次主题教育月活动的开展，将进一步推动学院教师深入思考教育初心，勇担育人使命，努力成为塑造学生品格、品行、品味的"大先生"。

活动期间，学院将通过"线上+线下"相结合的方式，采用专题报告、集中学习、研讨交流、经验分享等多种形式，引导教师深入领会新时代教师职业行为准则的具体要求。同时，学院还将评选表彰一批师德标兵和优秀教育工作者，发挥示范引领作用，营造崇德尚教的良好氛围，推动形成优良学风教风，为建设高水平职业院校贡献力量。

第一阶段：动员部署（3月上旬）

  • • 召开师德建设主题教育月活动动员大会
  • • 组织学习《新时代高校教师职业行为十项准则》等文件精神
  • • 制定活动实施方案，成立活动领导小组
  • • 开展师德师风建设专题调研，收集意见建议
  • • 组织全体教师签署师德承诺书

第二阶段：学习实践（3月中下旬）

  • • 邀请知名专家开展师德专题讲座
  • • 组织教师经验交流分享会，展示优秀育人案例
  • • 举办师德师风案例研讨会，深入剖析典型案例
  • • 开展"我心目中的好老师"主题征文活动
  • • 评选表彰优秀教师代表，树立先进典型

活动要求

思想引领

坚持以习近平新时代中国特色社会主义思想为指导，深入贯彻党的教育方针。全面落实立德树人根本任务，充分认识加强师德师风建设的重要性和紧迫性，切实提高政治站位，强化责任担当。

责任担当

牢记立德树人根本任务，争做"四有"好老师，做学生锤炼品格、学习知识、创新思维、奉献祖国的引路人。坚持教书和育人相统一，坚持言传和身教相统一，坚持潜心问道和关注社会相统一，坚持学术自由和学术规范相统一。

以身作则

严格遵守教师职业道德规范，以德立身、以德立学、以德施教，做学生健康成长的指导者和引路人。坚持教育者先受教育，以高度的责任感、使命感和荣誉感做好教书育人工作。

长效机制

建立健全师德建设长效机制，将师德师风建设要求贯穿教育教学全过程。完善师德考核评价机制，把师德表现作为教师考核、聘任和评优的首要标准。
